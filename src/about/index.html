Getting Started

It is very easy to use WebOS. Just follow these steps:

  1. Open a Web Browser. The latest version of Chrome or Edge is recommended.
  2. Visit the WebOS page.
  3. Try it out! Click around and explore the features.
  4. Install the PWA for a better experience (optional).

That's it! You can also use WebOS on you mobile device.

Please note that WebOS is still in development.
